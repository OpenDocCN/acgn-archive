## 下载

### Docker

```
docker pull apachecn0/acgn-archive
docker run -tid -p <port>:80 apachecn0/acgn-archive
# 访问 http://localhost:{port} 查看文档
```


### NPM

```
npm install -g acgn-archive
acgn-archive <port>
# 访问 http://localhost:{port} 查看文档
```
